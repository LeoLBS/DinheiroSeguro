## Dinheiro Seguro

---

### Sobre o projeto

O Dinheiro Seguro é uma aplicação simples desenvolvida em Java com o objetivo de auxiliar no controle de economia pessoal.

O sistema permite registrar valores que deixaram de ser gastos e acompanhar o total economizado em um determinado período.
A proposta é incentivar a consciência financeira através de um registro prático e organizado.

---

### Tecnologias utilizadas

**Versão 1.0**

- Java JDK 21
- Maven
- IntelliJ IDEA

---

### Funcionalidades

- Registro de valores economizados
- Cálculo do total acumulado
- Consulta do histórico de registros

---

### Pré-requisitos

- JDK 21 instalado
- Maven instalado
